ETIOLATION CALCULATION BREAKDOWN

How crowding modifies growth shape (tall/thin vs short/wide)
WHAT IS ETIOLATION?
Biological phenomenon: Plants grown in crowded conditions stretch tall and thin to compete for light. Isolated plants grow short and wide with strong stems.

In this simulation: Plants detect neighbors within their root radius and modify their growth proportions:
• More neighbors → Grow TALLER (2x length) and THINNER (0.5x width)
• No neighbors → Grow normally (1x length, 1x width)
CALCULATION PROCESS
STEP 1: Determine Root Radius
rootRadius = defaultRootSpreadDistance + (rootDistanceMultiplier × widthGrowth)
Default: 1.0 + (1.0 × widthGrowth)
As plant grows wider, its root zone expands → can detect more neighbors
STEP 2: Query Local Density
(neighborCount, totalBiomass, avgSize) =
  ManagerWorld.GetLocalDensity(position, rootRadius, plantId)
Asks ManagerWorld: "How many healthy plants are within my root radius?"
• Only counts Healthy state plants
• Excludes self from count
• Returns neighbor count and biomass data
STEP 3: Calculate Intensity
intensity = Clamp01(neighborCount / 5.0)
Normalize neighbor count to 0-1 range:
• 0 neighbors → intensity = 0.0 (no effect)
• 5+ neighbors → intensity = 1.0 (maximum effect)
• 3 neighbors → intensity = 0.6 (partial effect)
STEP 4: Calculate Length Modifier
lengthModifier = Lerp(1.0, etiolationStrength, intensity)
Default etiolationStrength = 2.0
• intensity = 0.0 → lengthModifier = 1.0 (normal)
• intensity = 1.0 → lengthModifier = 2.0 (double height)
• intensity = 0.6 → lengthModifier = 1.6 (60% taller)
STEP 5: Calculate Width Modifier
widthModifier = Lerp(1.0, 1/etiolationStrength, intensity)
Default 1/etiolationStrength = 0.5
• intensity = 0.0 → widthModifier = 1.0 (normal)
• intensity = 1.0 → widthModifier = 0.5 (half thickness)
• intensity = 0.6 → widthModifier = 0.7 (30% thinner)
STEP 6: Return Modifiers
return (lengthModifier, widthModifier)
These modifiers are multiplied into the growth formula:
• lengthGrowth = base × vigor × moisture × lengthModifier
• widthGrowth = base × vigor × moisture × widthModifier

GetEtiolationModifier() Method
(float lengthMod, float widthMod) GetEtiolationModifier(PlantSeed seedData)
{
  // Calculate root radius
  float rootRadius = defaultRootSpreadDistance +
    rootDistanceMultiplier × (plantWidth - defaultPlantWidth);

     // Query neighbors
  (neighborCount, _, _) = ManagerWorld.Instance.GetLocalDensity(
    seedData.bottomPosition, rootRadius, seedData.id
  );

     if (neighborCount == 0) return (1.0, 1.0);

     // Calculate intensity and modifiers
  float intensity = Clamp01(neighborCount / 5.0);
  float lengthMod = Lerp(1.0, etiolationStrength, intensity);
  float widthMod = Lerp(1.0, 1/etiolationStrength, intensity);

     return (lengthMod, widthMod);
}
KEY INSIGHTS: • Dynamic Detection: Root radius expands as plant grows wider → can detect more distant neighbors over time.

• Opposite Effects: Length and width modifiers are inverses - tall plants are thin, short plants are thick.

• Soft Cap at 5: Effect maxes out at 5 neighbors. More neighbors don't increase intensity further.

• Inspector Control: etiolationStrength parameter (default 2.0) controls how extreme the effect is.

• Emergent Patterns: Creates natural forest-like clustering (dense tall centers, spread wide edges).
