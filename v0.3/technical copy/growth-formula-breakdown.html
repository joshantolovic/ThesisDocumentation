GROWTH FORMULA BREAKDOWN

Mathematical decomposition of plant growth calculation
LENGTH GROWTH (Height/Vertical)
lengthGrowth = baseIncrement × seedVigor × moistureQuality × etiolationLength
1. Base Increment
Inspector: growthIncrementLength
Default: 0.002 units per cycle
Fixed growth rate set by designer. This is the baseline before any modifiers apply.
2. Seed Vigor
Assigned at plant birth (random)
Range: 0.7 to 1.3 (average: 1.0)
Inherent plant strength. Strong plants (vigor > 1.0) grow faster. Weak plants (vigor < 1.0) grow slower. This never changes after birth.
3. Moisture Quality
Calculated by GetMoistureQualityModifier()
Range: 0.5 to 1.0 (marginalMoistureGrowthRate to 1.0)
How close the available moisture is to optimal (default: 0.5).
• At optimal moisture → 1.0x growth
• At edge of acceptable range → 0.5x growth
• Outside range → Can't grow (drought begins)
4. Etiolation Length Modifier
Calculated by GetEtiolationModifier()
Range: 1.0 to etiolationStrength (default: 2.0)
Competition response - plants grow TALLER when crowded (reaching for light).
• 0 neighbors → 1.0x length (normal)
• 5+ neighbors → 2.0x length (stretched tall)
WIDTH GROWTH (Thickness/Roots)
widthGrowth = baseIncrement × seedVigor × moistureQuality × etiolationWidth
1. Base Increment
Inspector: growthIncrementWidth
Default: 0.0001 units per cycle
Fixed growth rate set by designer. Note: Much smaller than length increment (plants grow slower in width).
2. Seed Vigor
Same as length (shared value)
Range: 0.7 to 1.3
Same vigor value affects both dimensions equally.
3. Moisture Quality
Same as length (shared value)
Range: 0.5 to 1.0
Same moisture quality affects both dimensions equally.
4. Etiolation Width Modifier
Calculated by GetEtiolationModifier()
Range: 1.0 to 1/etiolationStrength (default: 0.5)
Competition response - plants grow THINNER when crowded (opposite of length).
• 0 neighbors → 1.0x width (normal)
• 5+ neighbors → 0.5x width (thin and stretched)
EXAMPLE: Crowded Plant in Good Conditions
Given:
• Base length increment = 0.002
• Seed vigor = 1.2 (strong plant)
• Moisture quality = 0.9 (near optimal)
• 3 neighbors detected → etiolation intensity = 3/5 = 0.6
• Length modifier = lerp(1.0, 2.0, 0.6) = 1.6
• Width modifier = lerp(1.0, 0.5, 0.6) = 0.7
Length growth: 0.002 × 1.2 × 0.9 × 1.6 = 0.003456 units
Width growth: 0.0001 × 1.2 × 0.9 × 0.7 = 0.0000756 units
Result: Plant grows 77% taller than normal but only 70% as thick → tall, thin plant stretching for light
KEY INSIGHTS: • Multiplicative System: All factors multiply together - a poor value in any factor drastically reduces growth.

• Etiolation Creates Shape: Same plant can be short/thick (isolated) or tall/thin (crowded) based purely on neighbors.

• Vigor is Permanent: Strong plants (high vigor) always have an advantage throughout their life.

• Width Affects Roots: rootRadius = base + (widthGrowth × multiplier) - wider plants have larger root zones.
