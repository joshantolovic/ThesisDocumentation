CRITICAL FLOW INSIGHTS: • Per-Plant Processing: Each plant evaluated independently every cycle - not event-driven.

• State Check First: Only Healthy plants can grow or accumulate drought time.

• Drought Accumulation: Only happens when CanGrow() fails (no suitable moisture).

• Growth = Multi-Factor: Base increment × 4 modifiers (vigor, moisture quality, etiolation length/width).

• Death Trigger: When drought timer exceeds threshold, spawns separate death coroutine.

• Infinite Loop: GrowthLoop runs forever (while true) - stopped only by scene change.
